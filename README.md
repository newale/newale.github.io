# Bienvenido/a

En este repo se encuentra el proyecto que sustenta mi página personal
El proyecto usa una arquitectura de monorepo que permite desplegar 3 aplicaciones:

- El sitio web principal (main-site)
- Una aplicación para administrar tareas pequeñas que guarda su estado en un _Local Storage_ (por-hacer)
- Una aplicación para calcular la escala de notas que tuve que usar el año pasado (escala-notas)

Todas están principalmente en un ecosistema javascript

El sitio web principal, usa eleventy para mantener el contenido en markdown y un templating sencillo
